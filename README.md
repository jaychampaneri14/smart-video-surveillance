# Smart Video Surveillance

> Motion detection, tracking, and alert system for security cameras

## Overview
This project is part of a comprehensive AI/ML portfolio covering computer vision python.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/smart-video-surveillance.git
cd smart-video-surveillance
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.

## Usage
```bash
python main.py
```
